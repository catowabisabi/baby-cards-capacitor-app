## Recipe — one command

After relaunching ComfyUI on `http://127.0.0.1:8188` (and verifying with
`curl http://127.0.0.1:8188/system_stats`):

```powershell
$env:PATH = "C:\Program Files\nodejs;$env:PATH"
cd "C:\Users\enoma\Desktop\opencode-work\agent-works\software\baby-cards"
.\scripts\resume-all-themes.ps1
```

The helper (`scripts/resume-all-themes.ps1`, `6726080` + `bdf1eb0`) runs all
8 remaining themes serially with per-theme `manifest refresh + atomic
per-theme commit`. Pre-flight `Test-ComfyUI` bails out if 8188 isn't
listening. Total wall time ~110 min unattended.

If you don't trust the helper, the manual equivalent per theme is:

```powershell
python scripts/comfyui_cards.py --theme <theme> --start 4 --count 100   # animals
python scripts/comfyui_cards.py --theme <theme> --start 1 --count 100   # cars / fruits / food / family / body / clothes / toys
npm run cards:manifest
git add "public/cards/<theme>/" "public/cards/manifest.json"
git -c core.autocrlf=false commit -m "feat(cards): regenerate <theme> with realistic style"
```

`animals` starts at `--start 4` because cards 1–3 (cat/dog/rabbit/hamster)
are kawaii and stay; cards 4–100 get replaced by realistic. All other themes
start at `--start 1` (full overwrite of the 3-card kawaii baseline).

Each batch is idempotent — it overwrites files in place.

## What to do if a batch crashes

- **`[Errno 22] Invalid argument` in `outer_sample → model_patcher.cleanup → callback`** →
  ComfyUI electron stdout-hijack regressed (e.g. desktop update). Re-apply both
  install-dir patches (`trange→range` in `sampling.py`, `print→pass` at KJNodes line 169).
- **CUDA OOM** → kill cached models, restart ComfyUI, retry.
- **`RuntimeError: 搵唔到 ComfyUI`** → server down; verify with
  `netstat -ano | findstr 8188` and `curl http://127.0.0.1:8188/system_stats`.
- **`OSError: [Errno 22]` from PIL `ImageDraw`** — possible, see `colors_renderer.py`
  `_pick_font`. Was once `'#fff60'` (string concat error), now uses `#FFFFFF`.

## Manual vs auto discovery

`scripts/comfyui_generate.py` and `scripts/comfyui_cards.py` use
`discover_server()` to ping `127.0.0.1:8188` → `127.0.0.1:8000` →
`localhost:8188` → `localhost:8000`. Override via:

- `--server HOST:PORT` (CLI flag, takes priority)
- `$env:COMFYUI_SERVER = "HOST:PORT"` (env var)
